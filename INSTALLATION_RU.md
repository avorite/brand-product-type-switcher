# Инструкция по установке и использованию

## Установка

1. Плагин уже находится в папке `/wp-content/plugins/brand-product-type-switcher/`
2. Активируйте плагин через меню **Плагины** в WordPress
3. Перейдите в **WooCommerce → Brand Type Switcher**

## Использование

### Шаг 1: Выбор брендов
- Установите галочки напротив нужных брендов
- Используйте "Select All" для выбора всех брендов сразу
- Рядом с каждым брендом показано количество товаров

### Шаг 2: Выбор типа товара
Выберите из списка:
- **С возможностью купить (Simple)** - обычные товары для покупки
- **Товар ссылка (External/Affiliate product)** - внешние/партнерские товары

### Шаг 3: Сохранение
- Нажмите кнопку **"Save Changes"**
- Начнется обработка товаров

### Шаг 4: Отслеживание прогресса
- Следите за прогресс-баром (процент выполнения)
- Просматривайте статистику (успешно/ошибки)
- Читайте логи операций в реальном времени

## Особенности

✅ **Сохранение Product URL**: При переключении типов товаров Product URL автоматически сохраняется и восстанавливается

✅ **Пакетная обработка**: Товары обрабатываются пакетами по 5 штук с задержкой 500мс между пакетами

✅ **Прогресс в реальном времени**: Видите процент выполнения и количество обработанных товаров

✅ **Подробное логирование**: Все операции записываются в лог с указанием успеха или ошибки

## Требования

- WordPress 5.0+
- WooCommerce 3.0+
- PHP 7.4+
- Таксономия `product_brand` должна существовать
